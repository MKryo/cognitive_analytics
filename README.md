# 因果帰納実験分析スクリプト

# setup
- Python用仮想環境ソフトウェアのインストール
```
pip install virtualenv
```

## 1. git clone
```
git clone fuga
```
## 2. virtualenv構築
```
virtualenv .
```
## 3. カレントディレクトリで以下のコマンドから仮想環境の有効化
```
source bin/activate
```

## 4. requirement.txtからpipでパッケージを入れる
```
pip install -r requirements.txt
```
## 5. 同期完了
```
pip list
```

- 無効化
```
deactivate
```
